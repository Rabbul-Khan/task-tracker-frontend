# Task Tracker

A simple and efficient Task Tracker app to manage, organize, and track your daily tasks and progress.

## Setup

Make sure to install dependencies:

```bash
# pnpm
pnpm install

```

## Development Server

Start the development server on `http://localhost:3000`:

```bash
# pnpm
pnpm dev
```

## Production

Build the application for production:

```bash
# pnpm
pnpm build
```

Locally preview production build:

```bash
# pnpm
pnpm preview
```
